In Your Corner Project

For this project, a new site had just been released and I was asked to look at the site and suggest changes that could make the user experience better

Problems

1. Logo was not noticable


2. Play symbol on 'See How It Works' button doesn't link to a video


3. Search bar gives the illusion that it is text input


4. Counselors specialties should not be in blue as blue is associated with links


Solutions

1. Enlarged logos


2. Removed play symbols so user is not expecting a video


3. Inserted an arrow so user knows it's a dropdown menu and not text input


4. Changed the titles to orange to add pop and not give the illusion of a link
